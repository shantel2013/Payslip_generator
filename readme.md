# Payslip Generator 

This is a Python-based project that generates PDF payslips for employees and emails them automatically.


- Generate Excel file with employee data
- Create individual PDF payslips using `fpdf`
- Email payslips using `smtplib` and Gmail


1. Clone the repository
2. Install dependencies:


- Enable 2FA on your Gmail account
- Create an App Password for this project
- Replace `'your_app_password'` in the code
